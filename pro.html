FAQs

Do I have to purchase a PRO license per Rails project?

Yes, the PRO license only applies to a single Rails project. That means you will have to pay for another license for your next Rails project using Maglev PRO.
Please email us if you need a bulk discount.

How does updates work?

By purchasing the Maglev PRO license, you will have access to our private Github repositories and private gems.

Can I use the UI kit I bought from X Y Z?

The awesome UI kit you purchased on TailwindUI won't work with Maglev PRO, not because of a technical issue but rather for legal reasons.

From the Tailwind UI site:

Creating a "website builder" project where end users can build their own websites using components or templates included with or derived from Tailwind UI.Almost all the licenses of those UI kits prevent the creation of a "site builder"

When the license is not clear about the "site builder" question, we highly recommend that you contact the people behind those UI kits and ask them for permissions.

Does the Maglev PRO include a theme by default?

Yes it does. Maglev PRO comes with our custom theme that we built specifically for this use case. The theme is a set of 50 sections with multiple variations and options.

How difficult is it to integrate Maglev PRO in an existing Rails project?

First of all, it's easier to use Maglev PRO from a brand new Rails project.

To give you a rough estimation an based on our last experience on a medium SaaS e-commerce Rails project (multi-tenants, multi-locales), it took us 3 days to setup Maglev and make it work within the application.
We spent another 3 days to build the first 5 minimal required sections (header, hero, list of products, contact form, footer).

If you give us access to your application code, we can give you an estimation of the work that has to be done to integrate Maglev PRO and build the first sections.

Can you assist us in providing a free SSL certificate for each site?

Although the Maglev PRO plan doesn't include a SSL certificates delivery solution, we've got enough experience in that field to help you find the best solution based on your hosting constraint and budget.

Do you offer refunds?

We don't offer automatic refund. However, in special cases, we might consider a partial refund.
